display:inline

Display inline : The element generates an inline-level box, which means it does not start on a new line and only takes up as much width as necessary.(height & width not affect). tags default inline display Examples are: span, a, strong

display:inline

display:inline-block

The element generates an inline-level box, but it behaves as a block-level element regarding its layout. It doesn't start on a new line and can have width and height properties.

display:inline-block

display:none

The element is not displayed at all, making it effectively invisible and taking up no space in the layout. This is often used for hiding and showing elements dynamically using JavaScript.
