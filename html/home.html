Relative, top:20px right:20px->Considers it's current position as a reference for positioning
Absolute, bottom:20px, right:30px->Considers, the closest parent element that is not static for positioning.Also if not specified, the width, then it fits upto the text within the content
Fixed, top:0px; left:20px;->Considers the viewport as reference for positioning.This is used widely in positioning navbars.Also it remains fixed even when the page is scrolled
Sticky,left:50px; top:10px; -> Remains fixed, upto a certain position, even if the page is scrolling. Combination of fixed+relative
